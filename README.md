# Capybara UI

## 🔎 Sobre
Bibliote de componentes UI para aplicações React.

Obs.: 
- Atualmente em versão beta (desenvolvimento), podendo haver alterações no nome da biblioteca e remoção de componentes.

## 💻 Instalação

```sh
npm install capybara-ui
```
